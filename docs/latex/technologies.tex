\subsection{Yocto}
\verb|Yocto| is an open source tool for building custom linux-based systems.
It is designed with embedded applications in mind.
It is widely considered to be the industry standard for \verb|IoT| Linux applications~\cite{yocto-manual}.

Yocto provides abstraction over hardware using \textbf{recipes}. A recipe
defines how to install a certain component or feature. Recipes can be reused,
which simplifies development by automating the installation process.

\subsection{LoRa}
\verb|LoRa| is a radio communication technique used for sending packets over long distances.
It is the basis for \verb|LoRaWAN|.\@

\subsection{CAN}
\verb|Control Area Network (CAN)| is a protocol commonly used both in automotive and industrial applications.
It is knows for being simple to wire and resistant to electrical interference.

\subsection{RS485}
\verb|RS485| is a serial protocol commonly used in industrial applications

\subsection{MQTT}
\verb|MQTT| is a lightweight publish-subscribe protocol commonly used in \verb|IoT| applications due to its low bandwidth requirements.
