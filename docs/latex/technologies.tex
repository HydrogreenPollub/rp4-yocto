\begin{itemize}
    \item[]
          \large\textbf{Yocto}\\
          Yocto is an open source tool for building custom linux-based systems.
          It is designed with embedded applications in mind.
          It is widely considered to be the industry standard for IoT Linux applications~\cite{yocto-manual}.

          Yocto provides abstraction over hardware using \textbf{recipes}. A recipe
          defines how to install a certain component or feature. Recipes can be reused,
          which simplifies development by automating the installation process.
    \item[]
          \large\textbf{LoRa}\\
          LoRa is a radio communication technique used for sending packets over long distances.
          It is the basis for LoRaWAN.\@
    \item[]
          \large\textbf{CAN}\\
          Control Area Network (CAN) is a protocol commonly used both in automotive and industrial applications.
          It is knows for being simple to wire and resistant to electrical interference.
    \item[]

    \item[]
          \large\textbf{MQTT}\\
          MQTT is a lightweight publish-subscribe protocol commonly used in IoT applications due to its low bandwidth requirements.
